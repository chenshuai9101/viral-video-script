# 短视频转爆款文案生成器

## 一、Skill概述

- **名称**：viral-video-script
- **版本**：v1.0
- **目标**：为自媒体创作者生成爆款短视频脚本
- **核心价值**：平台垂直适配 + 爆款公式内置 + 零门槛使用
- **适用人群**：自媒体创作者、短视频运营者、内容博主

## 二、核心功能

### 2.1 平台适配

| 平台 | 特点 | 时长 | 风格 | 核心要素 |
|------|------|------|------|----------|
| 抖音 | 快节奏、情绪化、强互动 | 15-60秒 | 爆点前置、悬念设置 | 3秒钩子、情绪峰值 |
| 小红书 | 种草风、生活化、情感共鸣 | 30-90秒 | 真实分享、干货价值 | 利他性、场景化 |
| 快手 | 接地气、真实感、强互动 | 30-120秒 | 老铁文化、情感连接 | 真实感、互动引导 |

### 2.2 爆款公式库

#### 标题公式
```
数字 + 痛点 + 承诺 = 高点击率标题
```

**爆款标题模板（100+）**：
- 「揭秘」类：`原来这才是...的真相`
- 「数字」类：`3个方法让你...`
- 「对比」类：`从...到...我只用了...`
- 「悬念」类：`没想到...竟然...`
- 「痛点」类：`还在为...烦恼？`
- 「蹭热点」类：`XX都在用的...`

#### 开头钩子库（50+模板）

| 类型 | 模板 | 适用场景 |
|------|------|----------|
| 震惊体 | `你绝对想不到...` | 知识科普、揭秘类 |
| 痛点体 | `你是不是也...` | 教程、解决问题类 |
| 数字体 | `只需要...就能...` | 干货、技巧类 |
| 冲突体 | `明明...却...` | 观点输出、情感类 |
| 身份体 | `作为...我必须说...` | 经验分享、专业类 |

#### 内容结构库（20+剧情模板）

**黄金结构（适合抖音）**：
```
1. 开头：3秒钩子抓住注意力
2. 铺垫：建立情境/痛点（5秒）
3. 转折：意外/反转/新知（10秒）
4. 高潮：核心爆点/价值输出（15秒）
5. 结尾：行动召唤/互动引导（5秒）
```

**种草结构（适合小红书）**：
```
1. 痛点引入：引发共鸣
2. 方案呈现：解决方案
3. 效果展示：真实案例
4. 细节补充：实用建议
5. 推荐总结：引导行动
```

### 2.3 一键生成流程

**输入参数**：
- `topic`：视频主题/核心内容
- `platform`：目标平台（douyin/xiaohongshu/kuaishou）
- `style`：内容风格（干货/情感/搞笑/种草）
- `duration`：视频时长（15s/30s/60s）

**输出内容**：
- 完整脚本（含分镜描述）
- 爆款标题（3个备选）
- 标签推荐（5-10个）
- 互动引导话术

## 三、API接口

### 3.1 生成脚本
```
generate_script(topic, platform, style, duration)
```

### 3.2 平台适配
```
adapt_platform(script, target_platform)
```

### 3.3 标题生成
```
generate_titles(topic, platform, count=3)
```

### 3.4 标签推荐
```
generate_tags(topic, platform, count=10)
```

## 四、实现方案

### 4.1 爆款公式库
```
/scripts/formulas.py
  - title_templates: 100+爆款标题模板
  - hook_templates: 50+开头钩子模板
  - structure_templates: 20+内容结构模板
```

### 4.2 平台特性库
```
/scripts/platforms.py
  - 抖音：douyin_config
  - 小红书：xiaohongshu_config
  - 快手：kuaishou_config
```

### 4.3 脚本生成器
```
/scripts/generator.py
  - ScriptGenerator 类
  - 支持自定义模板
  - 支持批量生成
```

## 六、使用示例

### 示例1：生成抖音爆款脚本
```python
from generator import ScriptGenerator

gen = ScriptGenerator()
script = gen.generate_script(
    topic="职场沟通技巧",
    platform="douyin",
    style="干货",
    duration="30s"
)
```

### 示例2：跨平台适配
```python
from generator import ScriptGenerator

gen = ScriptGenerator()
adapted = gen.adapt_platform(
    original_script="...",
    target_platform="xiaohongshu"
)
```
